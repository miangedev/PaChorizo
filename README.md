## Descripción

PaChorizo es un sitio web estático basado en un diseño original de tienda online de Julio César, replicado y modificado para presentar un negocio de chorizos. El propósito es académico y la interacción es mínima, sin funcionalidades de comercio electrónico reales.

## Demo en línea

Prototipo público disponible en:

## Instalación local

```bash
git clone https://github.com/miangedev/PaChorizo.git
cd PaChorizo
open index.html  # o ejecutar: python -m http.server 8000
```

Luego abre `http://localhost:8000` en tu navegador.

## Despliegue

El sitio está alojado estáticamente en `miange.online`, sin backend, correspondiente a un trabajo académico.

## Contribuciones

Este proyecto es un prototipo académico; si deseas sugerir mejoras visuales o de contenido, puedes:

1. Hacer un *fork* del repositorio.
2. Crear una rama con tus cambios (`git checkout -b mejora-x`).
3. Hacer commit y subir los cambios.
4. Abrir un *Pull Request* con detalles.
